How Siteswap Works

Each digit tells you how many beats a ball stays in the air. 3 = standard 3-ball cascade. Odd numbers cross to the other hand; even numbers return to the same hand. 0 = empty beat, 1 = a quick handoff, 2 = a hold. Try patterns like 441, 531, 97531, or b (hex for 11). The simulator uses real parabolic gravity arcs — taller throws take longer, just like real life.

Passing mode uses 4-handed siteswap: beats alternate between juggler A and B. Odd throws cross between jugglers (passes); even throws stay with the same juggler (selfs). Try 7 for a classic 6-club pass, or 786 for 3-count.
